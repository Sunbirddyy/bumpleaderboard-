# bumpleaderboard-
A modern Discord bot that automatically tracks DISBOARD bumps, generates beautiful leaderboard and provides detailed bump statistics for your server.


**What do you need to install**
npm install discord.js dotenv
npm install discord.js dotenv canvas

Don't forget to deploy slash commands 
node deploy-commands.js

Don't forget to create .env 

TOKEN=
CLIENT_ID= 

Without the ""

Enjoy !!
